📦 BƯỚC 1: CÀI ĐẶT
-------------------------------------------------------------------
1. Cài đặt thư viện:
   pip install -r requirements.txt

2. Cài đặt browser cho Playwright:
   playwright install chromium

✅ Xong! Giờ có thể dùng tool.


🚀 BƯỚC 2: SỬ DỤNG
-------------------------------------------------------------------

🔸 CÁCH 1: Đơn giản nhất
   python tiktok_counter.py
   
   → Chương trình sẽ hỏi URL, bạn chỉ cần paste link kênh TikTok vào.

🔸 CÁCH 2: Truyền URL luôn
   python tiktok_counter.py https://www.tiktok.com/@username

🔸 CÁCH 3: Lưu kết quả ra file
   python tiktok_counter.py https://www.tiktok.com/@username --save-json --save-csv
   
   → Sẽ tạo 2 file:
      - tiktok_stats_[tên]_[time].json (dữ liệu đầy đủ)
      - tiktok_videos_[tên]_[time].csv (mở được bằng Excel)

🔸 CÁCH 4: Load nhiều video hơn
   python tiktok_counter.py https://www.tiktok.com/@username --max-scrolls 50
   
   → Scroll 50 lần thay vì 20 (mặc định)
   → Càng nhiều scroll = càng nhiều video = càng lâu

💡 MẸO VẶT
-------------------------------------------------------------------
• Nếu muốn xem browser chạy:
  python tiktok_counter.py URL --headless

• Nếu muốn load TẤT CẢ video (rất lâu):
  python tiktok_counter.py URL --max-scrolls 100

• Nếu bị lỗi "không tìm thấy video":
  - Chạy lại với --headless để xem
  - Kiểm tra URL có đúng không
  - Thử VPN nếu bị block

• File CSV có thể mở bằng Excel để phân tích dễ hơn


⚠️ LƯU Ý QUAN TRỌNG
